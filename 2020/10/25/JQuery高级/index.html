#JQuery 高级

核心记忆点：
$(“”).show();
$(“”).on(“click”,fn{});
$(“”).each();
$.fn.extend({
max:function(a,..){},
min:function(a,..){},
});

##1 动画
1.三种方式显示和隐藏元素
1.show([speed,[easing],[fn]]) 显示隐藏的匹配元素。
参数：
speed:三种预定速度之一的字符串(“slow”,”normal”, or “fast”)或表示动画时长的毫秒数值(如：1000)

        easing:(Optional) 用来指定切换效果，默认是"swing"，可用参数"linear"
                *swing:动画执行的效果是 先慢，中间快， 最后又慢   *摆动
                *linear：动画执行效果是 匀速                    *线状的

        fn:在动画完成时执行的函数，每个元素执行一次。

    2.hide([speed,[easing],[fn]])   隐藏显示的元素
    3.toggle([speed,[easing],[fn]])   隐藏显示的元素

    **滑动效果
        1.slideDown([speed],[easing],[fn])
        2.slideUp([speed,[easing],[fn]])
        3.slideToggle([speed],[easing],[fn])
    **淡入淡出效果
        1.fadeIn([speed],[easing],[fn])
        2.fadeOut([speed],[easing],[fn])
        3.fadeToggle([speed,[easing],[fn]])

2.遍历
    1.js的遍历方式
        *for（初始化的值；循环结束条件；步长）
    2.jq的遍历方式
        1.jq对象.each（callback）
            *结束循环：
                如果当前function返回为false，则结束循环。同break
            *结束当前循环：
                如果返回为true，则结束本次循环，继续下次循环。同continue
            *获取index   回调函数内声明index和element
                *citys.each(function (index,element) {
        2.$.each(object, [callback])
            注意点同1
        3.for..of;
            *eg：
             for(li of citys){
                   alert(li.innerHTML)
                }

3.事件绑定
    1.jquery标定的绑定方式
        jq对象.事件方法（回调函数):
        *注：如果调用事件方法，不传入回调函数，会触发浏览器默认行为
            如：$("#name").focus();     //如果不绑定事件，该方法让文本输入框自动获得焦点
        *表单对象.submit();  //让事件提交
    2.on的方式绑定事件，off的方式解除绑定
        jq对象.on（”事件名称“，回调函数）
        jq对象.off（”事件名称“）
            *jq对象.off（）； //会解除组件上所有事件绑定
    3.事件切换：toggle
            jq对象.toggle（fn1,fn2 ...)
                *当jq对象对应的组件被点击后，会执行fn1，第二次点击执行fn2...
                当最后一个方法被执行完后，会再执行fn1
        *1.9版本 .toggle() 方法删除,jQuery Migrate（迁移）插件可以恢复此功能。


4.插件:增强部分JQuery的功能
    1.实现方式：
        1.$.fn.extend(object) 
            *增强通过JQuery获取的对象的功能。  如：$(“#a”).max();

            *定义多个方法，方法之间用 ， 连接
            *格式如下：
                 $.fn.extend(
                    {
                    max:function (a,b) {
                    },
                    min:function (a,b) {
                    },

                    }
                );

        2.$.extend(object) 
            *增强$对象本身             如：$.max();
